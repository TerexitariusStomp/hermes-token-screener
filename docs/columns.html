Token Scoring Formula

FactorPointsDescription
FDV/Volume Ratio0-25Low FDV + high volume = high opportunity. Ratio >2 = 25pts
Channels + Mentions0-20More channels mentioning = more legitimate discovery
Smart Wallets0-15More smart money wallets = stronger signal (50+ = 15pts)
Dev Holding0-10Developer still holding tokens = conviction signal
Social Signals0-10Twitter sentiment + Telegram social score
Price Momentum0-10Positive price movement across h1/h6/h24

Multipliers: burned LP (+15%), CTO (+10%), smart wallets >20 (+15%), verified contract (+20%)

Penalties: h1 < -60% (×0.1), death spiral (×0.3), honeypot (score = 0)

Wallet Scoring Formula

FactorPointsDescription
Realized PNL0-35Profit TAKEN, not paper gains
Trade Count0-20Active wallets = established traders
Win Rate0-10Profitable tokens / total tokens
ROI0-10Average profit per token
Entry Timing0-8Earlier entries = better score

Smart Tag0-5TOP1, KOL, SMART = higher score
